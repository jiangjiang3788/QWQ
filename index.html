                  Prompt / 注入诊断
                  

                  

                  

                  设计预览用于核对预算与顺序；最近真实 Prompt 来自最近一次私聊请求；世界书命中诊断显示实际候选、命中原因与输出字符。世界书开关与预算会作用于真实请求；结构化档案、最近聊天和状态栏在此处仅用于设计预览，不会替代角色设置中的长期记忆三选一模式。AI 请求诊断不会展示 API Key，并可查看排队、首字节、流完成与取消原因。
