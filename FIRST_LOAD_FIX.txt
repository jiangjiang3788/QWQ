QWQ V16.1 首次加载修复说明

修复内容：
1. 移除阻塞首屏解析的 Google Fonts 外链样式。网络无法访问 fonts.googleapis.com 时，旧版本会卡住后续脚本；已改为使用现有系统字体回退。
2. 修复 API 设置页 MutationObserver 自触发死循环。旧逻辑监听整个设置页，同时在回调中反复改写自己监听区域的文本，导致全新浏览器首次初始化时主线程被占满。
3. 为修复后的 API UI 脚本增加缓存版本参数，确保部署后浏览器获取新文件。

验证：
- 全新 Chromium 用户数据目录
- 空缓存
- 空 IndexedDB
- 本地 HTTP 冷启动
- 应用成功输出：[Startup] 单用户模式初始化完成
- 静态检查与全部 JavaScript 语法检查通过
