# V2.9-R0 重构基线

本版只建立可安全继续重构的基线，不修改结构化记忆数据格式。

## 已完成

- 首页应用入口由 App Registry 统一生成。
- 新增独立“角色”App。
- “记忆”成为一级 App，并提供角色选择。
- 删除空电话页、空存储页和旧更新弹窗。
- 底部导航由四项收敛为联系人、聊天、Apps 三项。
- 隐藏群聊创建按钮从生产 HTML 删除，群聊旧数据兼容代码暂时保留。
- 生产包删除历史版本说明、旧验证报告和旧模板副本。
- 增加 Feature Flags、发布白名单工具和基线检查。

## 未改变

- 记忆模板 schema 仍为 V2.8。
- 8 张表、209 条 rows 记录不变。
- 主聊天、总结 API、向量 API 行为不变。
- V2.1 至 V2.8 的记忆测试继续保留。

## 下一阶段

V2.9-R1 将拆解 `memory_table.js`：

- domain：数据规范、生命周期、策略。
- services：更新、检索、审核、反馈和任务。
- adapters：数据库、API、导入导出。
- ui：记忆、待处理、管理三个工作区。

在 R1 完成前，不删除 `window.MemoryTable*` 兼容接口。
