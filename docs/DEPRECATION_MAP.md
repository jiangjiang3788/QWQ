# 废弃与兼容清单

## R0 已从生产 UI 移除

- 电话空页面及其底部导航入口。
- 空 `storage-screen`。
- 隐藏的群聊创建按钮。
- 旧版本更新弹窗。

## 暂时保留兼容代码

- 群聊设置 DOM 与历史数据分支。
- 旧日记模式。
- 思维链、正则、状态栏等高级工具。
- 电量互动、保活和悬浮工具。
- `window.MemoryTable*` 全局接口。

## 删除流程

1. 先通过 Feature Flag 关闭入口。
2. 保留至少一个版本的数据导出能力。
3. 记录真实调用与用户数据情况。
4. 确认无依赖后删除运行代码。
5. 运行全部回归和模板数据对比。
