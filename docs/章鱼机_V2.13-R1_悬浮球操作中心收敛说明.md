# 章鱼机 V2.13-R1：悬浮球操作中心收敛说明

本版对悬浮球的信息架构做一次性收敛，不增加新的并行页面。

- 删除顶部 `Proment` 入口。Prompt 设计与核对继续由独立 App 负责。
- 将顶部 `操作` 与 `详情` 合并为 `操作记录`。主页面负责状态与历史，点击记录进入同一工作区内的详情。
- 删除 `普通 / 详细 / 高级` 查看级别。悬浮球固定展示实用详情：执行阶段、后台工作、数据变化、记忆核验、模型请求与可解释来源。
- 原始 Provider 请求、完整合并 Prompt 和高级结果 JSON 不再出现在悬浮球中。
- 报告导出固定使用脱敏的 detailed 协议，避免用户在多个层级之间做无意义选择。
- 顶部操作由 8 项缩减为 6 项，桌面六列，移动端三列，窄屏两列。
