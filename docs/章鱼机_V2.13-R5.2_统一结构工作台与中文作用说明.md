# 章鱼机 V2.13-R5.2 统一结构工作台与中文作用说明

## 一、本版目标

本版处理三项收敛：

1. 空输入框连续敲击 3 次 Enter 即请求 AI 回复，继续保留 30 秒触发冷却；
2. 记忆行“作用”在所有编辑界面使用中文标签；
3. 表结构编辑器不再拆成“字段 / 表格 / 结构 JSON”三个并列视图，改为一个全屏统一工作台。

## 二、连续 3 次 Enter

- 统计窗口：3 秒；
- 触发次数：3 次独立按键；
- 冷却时间：30 秒；
- 长按 Enter 不计为多次；
- Shift、Ctrl、Meta、Alt 组合 Enter 不参与；
- 输入框有文字时保持普通发送行为；
- AI 正在回复时不会重复触发。

## 三、作用中文化

界面显示：

- `fact` → 已确认事实
- `temporary_state` → 临时状态
- `soft_preference` → 柔性偏好
- `hard_boundary` → 明确边界
- `reminder` → 提醒事项
- `historical_context` → 历史背景
- `candidate` → 未审核候选

为保证旧数据、召回规则和生命周期逻辑兼容，数据内部仍保存英文键。中文输入也会自动映射回对应英文键。

覆盖位置：

- 全屏整行编辑；
- 表格行内标签编辑；
- 记忆详情侧栏标签编辑；
- 旧提示式作用编辑入口。

## 四、表结构编辑器收敛

### 旧结构

- 字段视图：只编辑字段基础项，高级字段另放折叠区；
- 表格视图：只编辑表格基础项，高级策略另放下方区域；
- 结构 JSON：再次把同一 schema 按路径重复列出。

### 新结构

编辑器使用整个屏幕，只保留一个连续工作台：

1. 模板名称、模板描述和数量摘要；
2. 全部表格设置表；
3. 当前选中表的字段设置表；
4. 页面底部折叠的原始 JSON 导入和排错出口。

不再提供三个并列标签页，不再出现三处重复编辑入口。

## 五、表格设置列

基础项和高级项在同一行：

- 表格名称、模式、层级；
- 自动更新、触发方式、轮次间隔、消息间隔；
- 读取上限、重叠消息；
- 更新 API；
- 允许新增、允许修改、允许删除；
- 注入方式、Top-K、阈值、字符预算、有效期；
- 包含置顶、包含完成；
- 提取规则、更新附加规则、注入附加规则；
- 字段数量和排序、删除操作。

## 六、字段设置列

基础项和高级项在同一行：

- 分组、字段名、类型、默认值；
- 选项、最小值、最大值；
- 普通显示、AI 编辑；
- 摘要标签、显示格式；
- AI 提示、条件规则；
- 排序和删除操作。

选项、最小值、最大值均不是必填项。普通文本字段可以保持为空；进度字段在最终规范化时仍沿用系统默认的 0–100。

## 七、兼容性

- 不修改模板、表格、字段或行 ID；
- 不迁移正式记忆数据；
- 不改变 schema 的实际存储结构；
- 原始 JSON 仍与可视化表格使用同一份草稿；
- 旧英文作用键继续有效；
- V2.13-R5 和 R5.1 的候选闭环、原子晋升、全屏 KV 编辑和按记录计数能力全部保留。
