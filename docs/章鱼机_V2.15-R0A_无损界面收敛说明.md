# 章鱼机 V2.15-R0A 无损界面收敛说明

## 范围

本版只收敛界面和入口，不迁移、不删除记忆表、字段或正式记忆。

## 写入方式界面

普通表只显示：

- 自动生效
- 待确认
- 仅人工编辑

底层 `review` 与 `candidate` 仍分别保留。已有模式在界面统一显示为“待确认”；用户再次选择“待确认”时，原来是审核的仍保持审核，原来是候选的仍保持候选。长期候选职责额外显示“长期晋升”，其他表不显示。

## 字段实际分流

表格设置显示字段最终路由数量：直接写入、待确认、仅运行态、仅人工和禁止自动写入。审核与候选同时存在时会在“待确认”后显示底层明细。

## 应用注册表

桌面和 Dock 不再维护平行 ID 数组。每个应用在唯一注册项中声明 `placement.home` 或 `placement.dock` 与顺序，启动器据此自动生成。Proment 的桌面位置与应用定义放在同一处。

## 记忆诊断中心

管理首页只保留“表结构编辑器”和“记忆诊断中心”两个主要入口。诊断中心集中进入：结构完整性、来源与时效、召回与索引、质量测试、历史与回滚。底层模块保持独立。

## 维护作业

原“任务队列”用户文案改为“维护作业”。内部模块名、持久化结构和任务状态不变，避免无意义的数据迁移。
